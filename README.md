# Text-RPG Eldritch Chronicles: The Forgotten Realms

This is a text-based Role-Playing Game (RPG) set in the "Eldritch Chronicles" universe within the "Forgotten Realms" setting. The game provides players with an immersive storytelling and gaming experience without the need for graphical elements.

## Getting Started

1. Clone the repository:

```
git clone https://github.com/Baelrin/Text-RPG-Eldritch-Chronicles-The-Forgotten-Realms.git
```

2. Navigate to the project directory:

```
cd Text-RPG-Eldritch-Chronicles-The-Forgotten-Realms
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the game:

```
python master.py
```

## Dependencies

- Python 3.x
- aiohttp
- cassandra
- pydantic
- SQLAlchemy
- numpy
- requests
- tqdm
